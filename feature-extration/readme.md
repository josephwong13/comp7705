### 1. Installation

If you don't have pipenv:
```
pip3 install pipenv
```

After installing pipenv:
```
pipenv install
```

### 2. (To use in jupyter notebook) Run

```
pipenv run jupyter notebook
```

### 3. (To use in VS code) Run
```
pipenv shell
```
